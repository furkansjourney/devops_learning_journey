# PostgreSQL Learning Notes

This section documents my learning process and hands-on practice with PostgreSQL.
The focus is on understanding core database concepts, writing efficient queries,
and gaining practical experience with relational data management.

## Topics Covered
- PostgreSQL architecture basics
- Database and table management
- Data types and constraints
- CRUD operations (SELECT, INSERT, UPDATE, DELETE)
- Indexes and performance basics
- Relationships and joins
- Backup and restore fundamentals

## Practice Approach
- Writing and testing SQL queries
- Designing small sample schemas
- Practicing query optimization
- Running PostgreSQL locally and via containers

## Goal
Build a strong foundation in PostgreSQL to support backend systems and
DevOps workflows where databases are a critical component.
