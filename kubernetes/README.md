# Kubernetes Learning Notes

This section tracks my Kubernetes learning journey with a strong emphasis on
hands-on practice and CKA-oriented fundamentals.
Notes include concepts, YAML manifests, and practical exercises.

## Topics Covered
- Kubernetes architecture and core components
- Pods, ReplicaSets, and Deployments
- Services and networking basics
- ConfigMaps and Secrets
- Volumes and persistent storage
- Basic troubleshooting and debugging
- Resource management (requests & limits)

## Practice Approach
- Writing and applying YAML manifests
- Running local clusters (minikube / kind)
- Experimenting with common kubectl commands
- Debugging failing Pods and Deployments

## Goal
Gain practical Kubernetes skills and prepare for the
Certified Kubernetes Administrator (CKA) exam through consistent hands-on work.
